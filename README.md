# sunbridger-api-mv 
npm包发布的一个预设仓库

## NPM地址 ： https://www.npmjs.com/package/sunbridger-api-mv

### npm install sunbridger-api-mv  --save-dev

```
"devDependencies": {
    "sunbridger-api-mv": "1.0.0"
}
```

#### 形如下面方式编写即可
```
let pro = require('sunbridger-api-mv');
pro.then(data=>{
    console.log(data);
}).catch(e=>{
    console.log(e)
})
```
